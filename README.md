# Portfolio – Nolann Chauveaud

## Présentation

Ce projet est un **portfolio web personnel** réalisé dans le cadre du module **2FCSS – Frameworks CSS** à **Hexagone IT School**.

L’objectif est d’intégrer une maquette Figma de portfolio en utilisant un framework CSS, tout en respectant les principes de responsive design, de lisibilité du code et de bonnes pratiques front-end.

```

## Maquette Figma

## Sections du site

Le portfolio contient les sections suivantes :

* **Accueil** : présentation et liens vers mes réseaux
* **About Me** : parcours et profil étudiant
* **Skills** : compétences techniques, outils et langues
* **Portfolio** : présentation de projets codés et/ou designés
* **Contact** : formulaire de contact avec validation côté front
